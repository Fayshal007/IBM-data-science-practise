This repository contains my **practice work on accessing databases using SQL Magic in Jupyter Notebook**.  
The notebook demonstrates how SQL queries can be executed directly inside a Jupyter environment using magic commands.

In this practice, I explored:

- Connecting to databases inside Jupyter Notebook
- Using **SQL Magic (`%sql` and `%%sql`)**
- Creating tables
- Inserting data into tables
- Running SQL queries (SELECT, WHERE, etc.)
- Viewing query results directly in notebook cells
- Using Python Variables in your SQL Statement
- Assigning the Results of Queries to Python Variables
- Converting Query Results to DataFrames
